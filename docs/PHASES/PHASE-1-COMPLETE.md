## API expansion (commit `427334c`)

| Endpoint | Description |
|---|---|
| `PATCH /me` | Update full_name, role, language_pref, biometric_enabled, university_id, major, department |
| `POST /me/verification` | Submit ID image (URL or base64), sets verification_status='pending' |
| `POST /push/register` | Store APNs / Web Push token, dedup on token |
| `DELETE /push/register` | Unregister token |
| `GET /catalog` | Wired through `db/catalogRepo.ts` (Supabase if configured, fallback fixture otherwise) |
| `GET /products/:id` | Product detail with size/sugar/ice options |
